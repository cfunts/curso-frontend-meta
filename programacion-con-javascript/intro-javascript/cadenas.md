Aquí tienes unas notas relevantes del texto sobre strings en JavaScript:

1. **Definición de String:**
   - Un string en JavaScript es una colección de caracteres encerrados por comillas simples o dobles.
   - Los strings son uno de los tipos de datos más comunes en JavaScript.

2. **Sintaxis Básica:**
   - Un string vacío se puede crear con comillas simples (`''`) o comillas dobles (`""`).
   - Para crear un string no vacío, se pueden incluir letras, símbolos de puntuación y números dentro de las comillas.

3. **Delimitadores:**
   - Las comillas simples o dobles que rodean un string se llaman delimitadores porque delimitan el valor del string del resto del código.

4. **Limitaciones de los Strings:**
   - Los strings no pueden dividirse en varias líneas. Intentar hacerlo resulta en un error de sintaxis.
   - Para incluir una comilla dentro de un string que ya está delimitado por el mismo tipo de comillas, se debe usar la otra clase de comillas como delimitador externo.

5. **Ejemplo de Error Común:**
   - Al intentar usar una contracción como "it's" dentro de un string delimitado por comillas simples, JavaScript interpretará el apóstrofe como el final del string, resultando en un error.
   - La solución es usar comillas dobles para delimitar el string: `"it's"` en lugar de `'it's'`.

6. **Consistencia:**
   - Aunque es posible mezclar comillas simples y dobles, es mejor usar solo uno de los dos tipos para mantener la consistencia en el código.

7. **Recomendación:**
   - Practicar creando diferentes tipos de strings para familiarizarse con su uso y comportamiento en JavaScript.
